# User Stories

## User Goal 1: Pantry Management

- As a user, I want to add food items to my pantry inventory so I can keep track of what I currently have.
- As a user, I want to update food quantities and expiration dates so my pantry information stays accurate over time.
- As a user, I want to remove foods that have expired or already been used so my inventory stays organized.
- As a user, I want to track expiration dates so I can see which foods will expire soon.

## User Goal 2: Meal Planning

- As a user, I want recipe recommendations that prioritize ingredients nearing expiration so I can reduce food waste.
- As a user, I want to view recipe detail pages containing ingredients, preparation instructions, and cooking times so I can plan meals better.
- As a user, I want smart shopping suggestions based on foods I already own so I can avoid unnecessary grocery purchases.
- As a user, I want to separately track meal-prepped foods so they do not go to waste.
